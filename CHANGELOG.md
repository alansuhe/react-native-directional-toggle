# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Added comprehensive documentation with badges and API details
- Added `engines` field to package.json (Node.js >= 18)
- Added `sideEffects: false` for better tree-shaking
- Added additional keywords for better discoverability
- Added CHANGELOG.md
- Added SECURITY.md
- Added feature request issue template
- Added pull request template

### Changed

- Updated GitHub Actions to use pnpm instead of yarn
- Improved package.json exports for better ESM/CJS compatibility
- Fixed lefthook.yml configuration
- Updated CONTRIBUTING.md to reflect pnpm usage

### Fixed

- Fixed CODE_OF_CONDUCT.md contact email placeholder

## [0.1.1] - 2025-XX-XX

### Added

- Initial release of react-native-directional-toggle
- Multi-option toggle component with horizontal and vertical layouts
- Smooth animations powered by react-native-reanimated
- Gesture support via react-native-gesture-handler
- TypeScript support with full type definitions
- Example app demonstrating usage
- CI/CD pipeline with GitHub Actions
- ESLint and Prettier configuration
- Conventional commit enforcement with lefthook
